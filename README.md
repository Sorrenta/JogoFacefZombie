# Como jogar:
Utilize as ⬅️⬇️⬆️➡️ de seu teclado para movimentar seu personagem, e o 🖱️ para mirar e rotacionar. Mire nos zumbis e meta bala com o clique esquerdo !
Dica: (Eles podem morrer entre 1 a 5 tiros, descubra) 💀
